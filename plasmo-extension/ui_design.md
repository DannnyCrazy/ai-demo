# UI设计文档

## 注入组件
目标网页上的浮动或固定位置按钮。

### 视觉样式
- **按钮标签**: "爬取该页面数据"
- **位置**:
    - 选项A: 固定在屏幕右下角（浮动操作按钮样式）
    - 选项B: 如果存在稳定的选择器，插入到产品标题或主要操作区域附近
    - *决定*: 固定右下角 `z-index: 9999` 更安全，避免布局破坏，确保无论页面结构如何变化都可见
- **样式**:
    - 背景颜色: 主蓝色（例如，`#3b82f6` - Tailwind blue-500）
    - 文本颜色: 白色
    - 内边距: `10px 20px`
    - 边框圆角: `4px` 或 `8px`
    - 阴影: `box-shadow` 以从页面内容中分离
    - 悬停状态: 深蓝色

### 交互
1.  **空闲**: 按钮可见
2.  **加载/处理中**:
    - 将按钮文本更改为"正在爬取..."或"正在打包..."
    - 禁用按钮以防止双击
    - 如果可能，显示加载图标
3.  **成功**:
    - 按钮文本恢复原样或短暂显示"完成"
    - 浏览器下载开始
4.  **错误**:
    - 显示提示错误或Toast消息表示失败

## 弹出窗口（可选）
由于要求是在页面上显示按钮，浏览器操作弹出窗口对于核心流程并非严格必需，但将来可用于配置。我们将专注于内容脚本UI。
